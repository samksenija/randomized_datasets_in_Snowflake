# Randomized Datasets in Snowflake
These are the various different randomized datasets, generated in Snowflake, and are applicable for various dedicated purposes &amp; free use. This is dummy data, generated by usage of various random functions available in Snowflake, that is, this data does not reference nor uses any 'real' data. It is intended for testing & training purposes. The structure always can be updated by user to meet their specific needs, as this is just an baseline randomized dataset.
<br/>
All datasets are created in or by using Snowflake & Snowflake SQL.
## 1. Dummy Ledger
Dummy ledger is completely randomized ledger, that is it is dummy ledger with completely randomized data, generated using Snowflake SQL. It contains following dummy columns:
<br/>
<br/>
`"date" DATETIME,
  "account_description" VARCHAR(50),
  "reference" VARCHAR(10),
  "debit" FLOAT,
  "credit"  FLOAT`
<br/>
<br/>
Debit & credit side are simply mirrored, that is first debit side is created and then just mirrored by credit side.
<br/>
<br/>
